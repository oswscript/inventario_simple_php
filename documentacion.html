Módulos del Sistema

  • INICIO:

    Este módulo muestra datos relevantes de los datos registrados del sistema.

  • NUEVO PRODUCTO:

    Formulario para el registro de productos en el sistema

  • PRODUCTOS:

    Listado de los productos del sistema, permite las opciones de VER, EDITAR y BORRAR cada registro.

    Tambien tiene opciones especiales para AGREGAR o RESTAR unidades de los productos que seleccione con los CHECKBOX.

    El botón llamado BORRAR PRODUCTOS, permite eliminar uno, varios o todos los prpductos al mismo tiempo, selecionados con los CHECKBOX.

  • SUMAR PRODUCTO:

    Permite ver un listado de los productos registrados. Esta sección tiene la funcionalidad de sumar al stock nueva cantidad de elementos a los productos.

  • RESTAR PRODUCTO:

    Permite ver un listado de los productos registrados. Esta sección tiene la funcionalidad de restar al stock elementos a los productos. No puede restarse una cantidad superior a la cantidad existente

  • RESTAR PRODUCTO:

    Listado de los categorpias registradas en el sistema; permite las opciones de VER, EDITAR y BORRAR cada registro.

  • USUARIOS:

    Listado de los usuarios registrados en el sistema; permite las opciones de VER, CREAR, EDITAR y BORRAR cada registro.

    Roles de los Usuarios: Cada usuario registrado, tiene un rol asignado. Cada rol tiene unos permisos distintos de uso en el sistema.

    • Administrador: tiene permisos absolutos del sistema, puede usar cada módulo y funcionalidad existente.
    • Supervidor General: tiene permisos absolutos del sistema, puede usar cada módulo y funcionalidad existente; pero solo puede ver los usuarios(supervidores y empleados) creados por El, y no por Supervisores y Supervisores Generales.
    • Supervidor: No puede ver el módulo INICIO, y en el registro de usuarios únicamente puede registrar empleados. No puede ver el listado de empleados de otros supervisores. Puede utilizart el resto de los módulos normalmente.
    • Empleado: Este usuario puede registrar productos, agregar y sacar registros del stock de dichos productos, registrar categorías y editar sus datos de usuarios.
